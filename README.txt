This plugin makes it very easy to delete cart items. Using it, you can delete all cart items at once or just the selected ones! 

One click, done! simple, easy and that is all!

== Description ==
This plugin helps you to delete all your cart items with on click. It also allows you to select the cart items that you don\'t want to purchase and delete them with a single click! 

== Installation ==
Upload the entire plugin\'s folder to the /wp-content/plugins/ directory.
Activate the plugin through the ‘Plugins’ menu in WordPress.
